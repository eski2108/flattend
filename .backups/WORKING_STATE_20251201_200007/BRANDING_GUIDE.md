# Coin Hub X - Brand Guidelines & Assets

## Brand Colors

### Primary Colors
- **Cyan**: `#00F0FF` - Main brand color, used for CTAs and highlights
- **Purple**: `#A855F7` - Secondary brand color, used for accents
- **Dark Blue**: `#0F172A` - Background primary
- **Slate**: `#1E293B` - Background secondary

### Accent Colors
- **Success Green**: `#22C55E` - Buy, success states
- **Warning Orange**: `#FB923C` - Warnings, alerts
- **Error Red**: `#EF4444` - Errors, sell, danger
- **Gold**: `#FBBF24` - Premium features

### Neutral Colors
- **White**: `#FFFFFF` - Primary text
- **Light Gray**: `#E2E8F0` - Secondary text
- **Medium Gray**: `#888888` - Tertiary text
- **Dark Gray**: `#444444` - Disabled states

## Typography

### Font Stack
```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "Roboto", "Oxygen", 
             "Ubuntu", "Cantarell", "Fira Sans", "Droid Sans", "Helvetica Neue", 
             sans-serif;
```

### Font Weights
- **Regular**: 400 - Body text
- **Semi-Bold**: 600 - Subheadings
- **Bold**: 700 - Headings, buttons
- **Extra Bold**: 900 - Hero text, large titles

## Logo Usage

### Primary Logo
- **Format**: SVG preferred, PNG fallback
- **Minimum Size**: 120px width
- **Clear Space**: Minimum 20px padding on all sides
- **Backgrounds**: Use on dark backgrounds only

### Color Variations
1. **Full Color**: Cyan + Purple gradient
2. **Cyan Only**: For single-color applications
3. **White**: For dark backgrounds
4. **Black**: For light backgrounds (use sparingly)

### Primary Gradient
```css
background: linear-gradient(135deg, #00F0FF, #A855F7);
```

### Background Gradient
```css
background: linear-gradient(135deg, #0F172A 0%, #1E293B 100%);
```

